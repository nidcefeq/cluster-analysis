# House Price Cluster Analysis

## Описание проекта

Проект посвящен кластерному анализу объектов недвижимости на основе датасета о продаже домов. Цель работы — выявить группы объектов с похожими характеристиками и определить основные сегменты рынка недвижимости.

В ходе исследования были выполнены:

* предварительная обработка данных;
* анализ пропущенных значений и дубликатов;
* обработка выбросов;
* генерация новых признаков;
* стандартизация данных;
* обучение и сравнение нескольких алгоритмов кластеризации;
* визуализация и интерпретация полученных кластеров.

## Используемые технологии

* Python
* Pandas
* NumPy
* Scikit-learn
* Seaborn
* Matplotlib

## Датасет

Используется датасет недвижимости округа King County (США), содержащий информацию о:

* стоимости домов;
* площади жилых помещений;
* площади участка;
* количестве спален и ванных комнат;
* годе постройки;
* наличии вида на воду;
* состоянии объекта;
* географическом расположении и других характеристиках.

## Этапы работы

### 1. Предобработка данных

* удаление служебных столбцов;
* анализ пропусков;
* удаление дубликатов;
* исследование распределений признаков;
* обнаружение и удаление выбросов методом IQR.

### 2. Feature Engineering

Признак `yr_renovated` был преобразован в:

* `was_renovated` — наличие реконструкции;
* `years_since_renovation` — количество лет после реконструкции.

### 3. Кластеризация

Были исследованы следующие методы:

* K-Means
* Agglomerative Clustering
* Gaussian Mixture Model (EM)

Для оценки качества использовались:

* Silhouette Score
* Davies–Bouldin Index
* Dunn Index

### 4. Визуализация

Построены:

* гистограммы распределений признаков;
* boxplot для поиска выбросов;
* корреляционная матрица;
* пространственная карта объектов недвижимости;
* сравнение кластеров различных алгоритмов.

## Результаты

Оптимальное количество кластеров: **6**.

Наилучшие результаты показал алгоритм **K-Means**:

| Метрика              | Значение |
| -------------------- | -------- |
| Silhouette Score     | 0.205836 |
| Davies-Bouldin Index | 1.449117 |
| Dunn Index           | 0.005277 |

## Интерпретация кластеров

### Кластер 0

Недорогие дома небольшой площади и сравнительно низкого качества.

### Кластер 1

Современные дома улучшенного качества по умеренной цене.

### Кластер 2

Крупные и качественные объекты с высокой стоимостью.

### Кластер 3

Дома средней стоимости с наличием подвала.

### Кластер 4

Реконструированные дома средней ценовой категории.

### Кластер 5

Наиболее дорогие объекты с видом на воду и выгодным расположением.

## Основные выводы

Кластеризация позволила выявить основные сегменты рынка недвижимости и определить влияние таких факторов, как:

* площадь объекта;
* качество строительства;
* наличие реконструкции;
* географическое расположение;
* близость к водоемам.

Полученные результаты могут использоваться для сегментации рынка недвижимости и дальнейшего анализа стоимости объектов.
